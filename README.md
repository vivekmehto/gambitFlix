**GambitFlix** is a modern, high-performance Netflix-style movie streaming web app built using
**React, Redux Toolkit, Tailwind CSS, Firebase Authentication, and TMDB API.**

Users can browse movies, watch trailers, explore categories, and enjoy a fast, cinematic UI experience.

## ‚ú® Features

- **Netflix-style UI** with hero banners, dynamic sections, and movie rows.
- **One-click trailer playback** using YouTube player integration.
- **Live movie data** fetched from the TMDB API.
- **Shimmer Loading UI** for a smooth user experience while data loads.
- **Firebase Authentication** for secure login and logout.

- **Fully responsive design** optimized for all devices.
- **Reusable components** such as MovieList, MovieCard, and Shimmer.
- **Client-side routing** using React Router DOM.
- **State management with Redux Toolkit** for clean and scalable architecture.

## üöÄ Getting Started: Running Locally

Want to run this project on your local machine? Follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/vivekmehto/gambitflix.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd gambitflix
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    ```

4.  **Create a .env file in the root folder:**

    ```bash
    VITE_TMDB_API_KEY=your_tmdb_api_key
    VITE_FIREBASE_API_KEY=your_firebase_api_key
    ```

5.  **Start the development server:**

    ```bash
    npm run dev
    ```

6.  **Open in your browser:**
    Visit `http://localhost:5173` (or the URL provided in your terminal).
